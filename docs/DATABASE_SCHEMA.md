# Database Schema

## Updated Core Tables

The updated domain model uses Opportunities and Operations rather than generic event/trading language.

## opportunity_series

A named recurring or long-lived concept that can produce opportunities.

Suggested fields:

- id
- name
- description
- category
- recurring_pattern
- official_website_url
- primary_organiser_id
- default_venue_id
- active
- created_at
- updated_at
