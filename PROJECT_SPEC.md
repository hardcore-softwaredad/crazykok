# Drenthe Trading Opportunities

## Goal

Build a desktop-first web application for managing every possible trading opportunity for a mobile food vendor operating primarily in Drenthe, Netherlands.

The application is intended to become a long-term business intelligence platform rather than simply an event list.

Primary goals:

- Discover every trading opportunity.
- Track application deadlines.
- Store vendor documentation.
- Score events by profitability.
- Record historical trading performance.
- Recommend future events.

The application should be designed to grow over many years.

Target platform:

- macOS
- SQLite
- Local-first
- Runs in browser

No cloud services required.
